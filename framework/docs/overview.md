# Overview — AmazonSender

## Идея в одном абзаце
Минималистичная утилита для разовой/редкой массовой email‑рассылки: письмо создаётся в Notion, по кнопке отправляется через Amazon SES на базу подписчиков из Supabase, после чего в Notion фиксируется отчёт и ошибки.

## Критерий успеха
Рассылка проходит один раз без дублей (кроме теста), статусы подписчиков обновлены (bounce/complaint/unsubscribe), в Notion есть отчёт и таблица ошибок, скорость отправки ограничена (5–6 писем/сек).

## Роль пользователя
Один пользователь (автор рассылки, админ и разработчик). Сервис не публичный.

## Стек и деплой
- Стек: Node.js (с перспективой React‑интерфейса).
- Деплой: локально → Vercel через GitHub.
